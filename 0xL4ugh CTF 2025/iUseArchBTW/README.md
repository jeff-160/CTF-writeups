## iUseArchBTW  

<img src="images/chall.png" width=600>

The challenge file contains the words of `i use arch btw` repeated multiple times in a seemingly random order.  

<img src="images/dist.png" width=600>

On analysis, it seems to resemble the syntax of an esolang. A quick google search for "iusearchbtw language" will lead use to [this repo](https://github.com/overmighty/i-use-arch-btw).  

We can just install the interpreter and run the source file through it to get the flag.  

Flag: `0xL4ugh{1_us3_4rch_l1nux_btw_4nd_y0u_sh0uld_t00_p4cm4n_r0ck5}`
